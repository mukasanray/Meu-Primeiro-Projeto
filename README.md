# Meu Primeiro Projeto
# Delicious Burger 🍔

Landing page moderna e responsiva desenvolvida para uma hamburgueria fictícia chamada **Delicious Burger**.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📌 Funcionalidades

* Menu hambúrguer responsivo
* Sistema de navegação superior
* Sessão de combos e hambúrgueres
* Adicionais, molhos, bebidas e sorvetes
* Área de contato e sugestões
* Simulação de montagem de pedidos

## 🎯 Objetivo do projeto

Projeto desenvolvido para prática de desenvolvimento Front-End, responsividade e organização de interfaces modernas para web.

## 📱 Responsivo

Compatível com computadores, tablets e celulares.
